ITM 101 In-class Activity Day 2 - Part 2

Overview

Help your table finish installing the software. Then, build something as a team with the tools we installed and ChatGPT.

Deliverable: Build something from the tools we installed. Use ChatGPT - Be creative. Have fun. If you fail that is ok, turn in what you did. Turn in a screenshot, code, or description of what you learned/did/improved/deployed as a team. Describe your personal contribution.

  • Only one person has to turn in the code from the team.
  • Include a list of everyone's first and last names in a comment.
  • Each person still needs to submit a comment describing their personal contribution.

Examples:

  • What is Javascript? What is Node.js?
  • How do I make a new folder and a new JavaScript file on my Mac (or Windows)?
  • Please write some Node.js code that does ...
  • How do I ...?
  • How do I put JavaScript code using Visual Studio Code into a JavaScript file and run it?

If you ran out of time or missed class, submit a screenshot of Node working with "node -v" in your powershell or terminal.
