ITM 101 In-class Activity Day 2

Programming with Generative AI

Summary

Complete this assignment with a buddy or your team. Not only is Generative AI good at producing answers to questions, it can also write code in most programming languages. This activity will show you several different prompts that you can use to generate code for specific tasks. At the end of the activity, you will be required to solve a problem by generating your own prompt.

Approximate Time to Complete

60 minutes

Hello, World! in JavaScript

Open ChatGPT in your browsers and enter the following prompt:

Write a hello world program in JavaScript that runs in Node.js. Also, show me how I can run this program with Node.js

Adding context

ChatGPT remembers the information that you give it. For example, we can make sure that in the current conversation, it will always write JavaScript programs to be run with Node.js instead of in a web browser. The next prompt will make ChatGPT remember that it should write JavaScript code that will run in Node.js.

From now on, write all JavaScript so that it can be run from a terminal with Node.js

Add two numbers

Try this prompt

Please write a JavaScript program that will add two numbers together and display the result.

Then this one

Make sure that I can pass the two numbers as arguments, instead of local variables.

Using the examples provided by ChatGPT, you should do the following:

  • In VSCode, create a file called numbers.js and copy/paste the code into the file
  • You will then run the code with a command that looks like this: node numbers.js 2 2. The resulting output should be 4.

Troubleshooting Code with ChatGPT

Debugging is the process of finding and fixing mistakes (often called bugs) in your code. ChatGPT can also assist with this process.

Copy and paste this prompt and code into ChatGPT:

Is there something wrong with this function?
function multiplyNumbers(a, b) {
  return a / b;
}

Try this one too:

Are there any edge cases that could cause this code to crash?
function divideNumbers(a, b) {
  return a / b;
}

Troubleshooting Node Installation

Set-ExecutionPolicy -ExecutionPolicy Unrestricted
