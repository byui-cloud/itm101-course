Week 3 Day 1 In-class Activity

What is a Web Site?

Summary

This activity will demonstrate how websites function at a basic level. You will create a simple web page that is generated with ChatGPT and deploy it in the Google Cloud Shell.

Approximate Time to Complete

45 minutes

Website Structure

When you access a website, the files are stored in a way that the computer can understand and know where to find each file.

Try the following prompt with ChatGPT:

Tell me simply how a website’s files are organized?

What Is an index.html File?

In the root directory there is a file called index.html that exists for almost every website. Ask ChatGPT about it with the prompt below.

What is the importance of the index.html file?

Let’s create one to see what the code looks like. Use this prompt.

Show me a basic index.html

Look at the code to see how it is organized. Talk with a buddy about it.

Other Elements of a Website

HTML files act as the backbone to a website. Each HTML file can give us a static page to view content but not in the best way. To make a web page more interactive and pretty, a web page links to other files to generate the page.

Ask ChatGPT:

What types of files does an index.html file link to?

To start building our own website, we will just use HTML, but eventually we will use ChatGPT to generate more complex sites.

Create an HTML File

  1. Using VS Code on your laptop computer, create a new folder named google-cloudshell-webserver
  2. Create an index.html file.
    1. In your new google-cloudshell-webserver folder, create a new file named index.html
    2. Ask ChatGPT to write an HTML file for you that describes any topic that interests you.
    3. Copy and paste the HTML code from ChatGPT into your index.html file.
    4. Save your index.html file.

View your HTML File

Browsers, such as Chrome or Safari, are built to run code in several different programming languages.

What different programming languages can I run in my browser?

We can see that HTML can be run from our browser. Take your index.html file and view it in your browser.

It’s nothing fancy, but it is the base for how websites are created.

Deploy your HTML File

If you want other people to see the web pages that you create, you must store the HTML files on a computer that is a web server. During this course, you will put your HTML files on several different types of web servers. Today, you will put your HTML file on a web server in the Google Cloud.

  1. Download and save this server.js file into your google-cloudshell-webserver folder.
  2. In a browser window that is signed into your BYU-Idaho Google account, open the Google Cloud Platform console
  3. Press Ctrl-O to open the project picker.
  4. Click the "New Project" button in the upper right corner of the Project Picker window.
  5. Create a new project named google-cloudshell-webserver.
    1. Name your project "google-cloudshell-webserver".
    2. Select "byui.edu" from the Organization drop-down list.
    3. Click the "Create" button.
  6. Activate the the Cloud Shell by clicking the Cloud Shell icon () in the upper right corner.
  7. Authorize the Cloud Shell.
  8. Upload your server.js and index.html files to the Cloud Shell.
    1. Click the more menu items icon () that is in the title bar of the Cloud Shell terminal panel.
    2. Click the "Upload" button.
    3. Click the "Choose Files" tab.
    4. Upload both of your server.js and index.html files
  9. In the Cloud Shell terminal panel, run your server.js file by entering this command: node server.js
  10. Preview your index.html file in your browser.
    1. Click the Web Preview icon () in the title bar of the Cloud Shell terminal panel. Then click "Preview on port 8080." This will open a new tab in your browser.
    2. In the address bar of the new tab delete this text: "?authuser=0" and then press the enter key. This should open your index.html page in your browser tab.
