ITM 101 In-class Activity Week 4 Day 1

Google Firebase Website Using AI

Scenario Overview

In this scenario, you are preparing for an upcoming interview with a company that uses Firebase hosting. You need to create a personal portfolio webpage on the internet that you can share with the employer. The job description states that a minimum requirement is experience with Google Firebase Hosting. The preferred candidates will have used Node.js. We will use both.

Project Requirements:
  • Use Visual Studio Code as your text editor
  • Use an automated scalable web hosting solution
  • Use Google services if possible
  • Use Node.js to install firebase and test your code
  • The project name must include part of your first or last name
Project Overview
  • Login to https://chat.openai.com/ with an account (create one if needed)
  • Login to https://console.firebase.google.com using your BYU-Idaho username (usually the email with numbers ex: lastname038@byui.edu).
    • Troubles with login - go to my.byui.edu, click on “Profile” scroll down to “Campus Email” and use that email for the Google Firebase login instead
  • Hopefully, you already have node.js and Visual Studio Code installed on your computer. If not, ask Chatgpt how to install it.
  • Deploy a website to Firebase: 
    • 1 - Once logged in to Google, create a new project in Firebase
    • 2 - In Visual Studio Code, make a project folder called “firebaseproject” and then new folder called ‘public’ inside of it
    • 3 - Make an index.html file and a styles.css file in the public folder
    • 4 - Open the terminal in Visual Studio Code (you will run some firebase installation commands from the next step provided by ChatGPT)
    • 5-6 - Ask ChatGPT the following questions and follow the instructions for each:
      • 5 - “How can I make a simple firebase webpage that uses node.js?”
      • 6 - “Please give me code for my index.html file and a styles.css file that would be a nice portfolio page and link them.”
