Week 4 Day 1 In-class Activity

Firebase Website Using AI

Scenario

In this scenario, you are preparing for an upcoming interview with a company that uses Firebase Hosting. You need to create a personal portfolio web page on the internet that you can share with the employer. The job description states that a minimum requirement is experience with Google Firebase Hosting, and the preferred candidates will have used Node.js. In this assignment, you will use both.

Business Requirements

  • Marketing portfolio website to obtain jobs

Project Requirements

  • Use Visual Studio Code as your text editor
  • Use an automated scalable web hosting service
  • Use Google Web Services if possible
  • Use Node.js to deploy your web page from your computer to the hosting service
  • The project name must include your last name

High-Level Steps

  • Sign into your ChatGPT account: https://chat.openai.com/ (Create an account if needed).
  • You should already have Node.js and Visual Studio Code installed on your computer. If not, ask ChatGPT how to install them.
  • Deploy a website to Firebase:
    1. Sign into the Google Firebase console using your BYU-Idaho email address (the email address with numbers, for example: bos24011@byui.edu).
      • Trouble signing in? Open my.byui.edu, click “Profile”, scroll down to “Campus Email”, and use that email for the Google Firebase sign in
      • If that doesn’t work, use a personal Gmail account.
    2. After signing into the Firebase Console, create a new Firebase project named “LASTNAME-week4-day1”
    3. Using Windows File Explorer or macOS Finder, make a folder on your computer named “LASTNAME-week4-day1” and then open that folder.
    4. Using Visual Studio Code, open the folder that you created.
    5. Still using VS Code, open a new terminal and run the following commands:
      1. node -v
      2. npm install -g firebase-tools
      3. firebase login
      4. firebase init
      5. Pick the option: "Hosting: Set up deployment for static web apps" or depending on your version it might be Hosting: Configure files for Firebase hosting and (optionally) set up GitHub action deploys.
      6. Make sure to pick "Existing Project" that you created in your browser.
      7. When it asks for a 'public' directory, hit "enter". If you type something you might get a folder named that.
      8. When it asks for a single-page app, type "N".
      9. For Github automatic deploys, type "N".

      The firebase init command should have created a folder named “public” inside your LASTNAME-week4-day1 folder.

    6. Ask ChatGPT: Please write code for an index.html file and a style.css file that would be a nice portfolio page and link them.
    7. Using VS Code, make an index.html file and a style.css file in the public folder. Paste the code written by ChatGPT into those two files.
    8. In the VS Code terminal, run this command:
      • firebase deploy
      The firebase deploy command should have copied your index.html and style.css files to the Google Hosting server and should have printed a URL to the terminal.
      1. Copy that URL and open it in your browser which should open your index.html page in your browser.
      2. Send the URL to your one of your teammates. Can you teammate open your index.html in her browser?
    9. Ask ChatGPT: How can I use Node.js to deploy the index.html and style.css files to Google Firebase Hosting?

      Did ChatGPT answer with the same commands that you ran in the VS Code terminal?
