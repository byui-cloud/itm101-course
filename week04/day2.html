Week 4 Day 2 In-class Activity

Deploy to the Cloud with Your Team

In this activity, you will work with your team to deploy a project to the cloud. You will use the skills you have learned in this course to create a project that is feasible, has market potential, and can be built using a suitable technology stack.

Make something cool and deploy it to the cloud - Hackathon style!

Use Firebase, AWS, or any other cloud service you like. Use ChatGPT to help you build it.

Technology Stack Selection

Your team this week will work on a choosing a tech stack!

  • Why and How: Watch this video Picking The Right Technology Stack
  • Task: Consider what technology (programming languages, frameworks) you would use to build it and why. Where will you deploy it?
  • Business Deliverable: A 1-paragraph explanation of basic technology choices including programming languages and a 1-paragraph description of where you will deploy it for hosting to the web/cloud. How many users would you expect? How does that change your deployment strategy?
    • Team Activity Technology Ideas

    Example:

    • Tech Stack Selection: The project will use existing frameworks for building mobile apps, such as React Native. These will allow for fast development and cross-platform support. The deployment to the cloud will be using Google Firebase Hosting.
  • Tech Deliverable: Deploy your project to the cloud. Turn in the URL. You can use Firebase, AWS, or any other cloud service you like. Use AI - Be creative. Have fun. If you fail that is ok, turn in what you did. Turn in a screenshot, URL, or description of what you did/improved/deployed in the cloud as a team. Describe your personal contribution.
