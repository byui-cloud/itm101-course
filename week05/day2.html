ITM 101 Week 5 Day 2 In-class Activity

Deliverable: Build something new as a team or improve a previous project with Databases and ChatGPT - Be creative. Have fun. If you want, you can use the project from Day 1 and improve it. If you fail that is ok, turn in what you did. Turn in a screenshot, URL, or description of what you did/improved/deployed. Describe your personal contribution.

  • Only one person has to turn in the code from the team.
  • Include a list of everyone’s first and last names in a comment.
  • Each person still needs to submit a comment describing their personal contribution.
