ITM 101 Week 6 Day 2 In-class Activity

Make your Firebase project better as a team.

Deliverable: Build something new or improve a previous project in this class as a team implementing/improving security or authentication. Use ChatGPT - Be creative. Have fun. If you fail that is ok, turn in what you did. Turn in a screenshot, URL, or description of what you did/improved/deployed in AWS as a team. Describe your personal contribution.

  • Only one person has to turn in the code from the team.
  • Include a list of everyone’s first and last names in a comment.
  • Each person still needs to submit a comment describing their personal contribution.
