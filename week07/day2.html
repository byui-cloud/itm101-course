Week 7 Day 2 In-class Activity

Deploy an AWS MYSQL Database.

Overview

Use AWS Relational Database Service (RDS) to deploy a MySQL database and connect to it with MySQL Workbench.

Steps

  1. Use ChatGPT or another generative AI to learn about MySQL. Ask a generative AI these questions:
    1. What is a relational database?
    2. What is MySQL?
    3. Why do we need MySQL?
  2. Use this web page that ranks database software to answer these two questions:
    1. What is the most popular relational database?
    2. Where does MySQL rank?

  3. Follow the instructions in this MySQL DB Tutorial

    The tutorial is owned and maintained by Amazon. Unfortunately, there is a missing instruction in the tutorial. Before you use MySQL Workbench to connect to your new RDS MySQL Server, you must do the following:

    1. Use the AWS Console to view the details of your newly created RDS MySQL Server.
    2. Find and view the security group.
    3. Open port 3306 in the inbound rules of the security group.
  4. After you use MySQL Workbench to connect to your RDS MySQL Server, capture a screenshot of the Workbench to submit for this assignment.
  5. Use the AWS Console to delete the RDS MySQL Server. This will save your AWS budget.
