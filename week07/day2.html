ITM 101 In-class Activity Week 7 Day 2

Create something cool in AWS as a team.

Deliverable: Build something new or improve a previous project in this class as a team with AWS. Use ChatGPT - Be creative. Have fun. If you fail that is ok, turn in what you did. Turn in a screenshot, URL, or description of what you did/improved/deployed in AWS as a team. Describe your personal contribution.

  • Only one person has to turn in the code from the team.
  • Include a list of everyone in a comment.
  • Each person still needs to submit a comment describing their personal contribution.
