ITM 101 In-class Activity Week 8 Day 1

AWS Website Using an S3 Bucket

Overview

In this scenario, you found a job or internship that you want to apply for. In the job description, they have a MINIMUM requirement of AWS S3 knowledge. You noticed they PREFER candidates who have experience with hosting websites with S3 buckets and can deploy code to the bucket. To build your resume, you will put code into an S3 bucket and make the website public directly from the S3 bucket.

Business Requirements:

  • Auto scaling web system for many users

Project Requirements:

  • Deploy a web application to an AWS S3 (autoscaling)
  • Use AWS (Amazon Web Services) and AWS S3 Buckets
  • Open necessary firewalls and ports for the website to work for clients who do not have a login to AWS
