ITM 101 In-class Activity Week 8 Day 1

AWS Website Using Cloud9 to S3 Bucket

 

OVERVIEW

In this scenario, you found a job or internship that you want to apply for. In the job description, they have a MINIMUM requirement of Cloud 9 knowledge. You noticed they PREFER candidates who have experience with Amazon S3. They also want candidates that have experience with hosting websites with S3 buckets and can deploy code to the bucket. To build your resume, you will use Cloud 9 IDE to put code into an S3 bucket and make the website public directly from the S3 bucket.

 

Business Requirements:
  • Auto scaling web system for many users

Project Requirements:

  • Use Cloud 9 IDE (not autoscaling because it is in an EC2 instances you would have to scale or add resources to)
  • Deploy a web application to an AWS S3 (autoscaling) bucket from Cloud 9
  • Use AWS (Amazon Web Services) and AWS Explorer
  • Open necessary firewalls and ports for the website to work for clients who do not have a login to AWS
