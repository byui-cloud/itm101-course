What is AWS Amplify?

AWS Amplify automates the deployment of many systems in AWS and is similar to Firebase. You can upload a folder like you could to an S3 bucket, but it does all the S3 bucket set up and configuration for you. Amplify is a set of tools from Amazon Web Services (AWS) that helps front-end developers build full-stack applications on AWS. It integrates various AWS services and tools into a framework that abstracts away cloud infrastructure complexity. This allows developers to focus on the user experience (UX) while leaving application infrastructure to the cloud provider.

Part 1 - Everyone upload a webpage to Amplify

Amplify is very simple to use. You zip up a website into a folder on your computer. Open Amplify in your AWS console and select your zipped folder.

Everyone turn in your own Amplify link of your zipped up folder.
