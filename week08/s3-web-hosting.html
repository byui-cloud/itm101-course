Hosting a Static Website in AWS S3

  1. Open the AWS Management Console
    1. Sign into the AWS Academy Canvas instance
    2. Open your AWS Academy Canvas course
    3. Within the Modules section, click “Launce AWS Academy Learner Lab”
    4. Click “▶ Start Lab”
    5. When the red circle (AWS ●) turns green (AWS ●), click the green circle (AWS ●) which will open the AWS management console in a new tab
  2. Create and Configure an S3 Bucket
    1. From the AWS management console find and open the S3 service
    2. Select the nearest region (Oregon)
    3. Create an S3 bucket
      Use your last name in the bucket name.
    4. Enable static web site hosting
      1. From the list of your buckets, click your new bucket’s name to open that bucket
      2. Click the “Properties” tab
      3. Scroll to the bottom of the list of properties and edit and enable “Static website hosting”
    5. Allow all public access to the S3 bucket
      1. Click the “Permissions” tab
      2. Edit “Block public access (bucket settings)” and disable (uncheck) the option “Block all public access”
    6. Edit the bucket policy and add the following policy. Before saving the new policy, change Bucket-Name to your bucket’s name. 
      {
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "PublicReadGetObject",
            "Effect": "Allow",
            "Principal": "*",
            "Action": [
                "s3:GetObject"
            ],
            "Resource": [
                "arn:aws:s3:::Bucket-Name/*"
            ]
        }
    ]
}
  3. Create and Edit an HTML File
    (You may also need a CSS file and a JavaScript file.)
    1. From the AWS management console open your S3 buckets
    2. Create and save an HTML file and any needed CSS and JavaScript files or edit them locally in Visual Studio Code and then upload to S3.
  4. Upload the HTML File to Your S3 Bucket.
    (Also upload CSS and JavaScript files if you created them.)
    1. Select the region where you created your S3 bucket (Oregon)
    2. Expand the region
    3. Search for the S3 service
    4. Find and open your S3 bucket name and upload the HTML, CSS, and JavaScript files to your bucket
  5. Open the Hosted HTML File in Your Browser.
    1. Return to the S3 console
    2. Open your bucket and click the “Properties” tab
    3. Scroll down to the “Static website hosting” property and click URL for your bucket’s web site
    4. Your web page should open in a new tab
  6. Verify that Your Web Site Is Available When the Learner Lab Is Stoped Stop the AWS Learner Lab Open the Hosted HTML File in Your Browser.
    1. Close the AWS management console tab
    2. Click “■ End Lab” in the AWS Canvas course
    3. Refresh the tab that contains your open web page and verify that it loads correctly
