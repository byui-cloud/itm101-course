AWS Elastic Compute Cloud (EC2) Instances

A virtual machine (VM) is a software-based emulation of a physical computer, allowing multiple operating systems and applications to run concurrently on a single physical computer, as if each virtual machine were a separate physical machine. In week 7, we used AWS EC2 to launch a Linux server on an EC2 instance. This Linux server is a virtual machine. We launch virtual machines and Amazon manages the physical machines for us.
